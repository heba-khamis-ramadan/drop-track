**DropTrack** is a community-driven app that helps users **pin, discover, and share nearby water sources** such as public taps, and refill stations. Built with a **Node.js + Express.js backend**, **MongoDB** for geolocation storage, and a **Bootstrap + Leaflet.js frontend** for an interactive map experience.
